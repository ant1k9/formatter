### Formatter

💠 Use formatter to customize the output of your programs.

#### Installation and usage

```
$ cargo install --path .
$ formatter --help
$ formatter --header "┏━┓" --prefix "┃ " --suffix " ┃" --footer "┗━┛" ls
```
